# Chemistry Applet (Version 1.0.2)

This repository hosts an interactive chemistry applet designed for use in:
- Moodle (via URL or SCORM)
- Canvas (via embed or external tool)
- GitHub Pages (public hosting)

## Features
- Dimensional Analysis Engine
- Stoichiometry Engine (molar mass parser included)
- Gas Law Solver (dynamic interface)
- Dark Mode Toggle
- Modern Cool-Blue Theme
- Banner + Favicon Support

## Deployment
This site is deployed using GitHub Pages.

## SCORM Packaging
To use in Moodle as a SCORM package:
1. Include the following files in a folder:
   - index.html
   - chemistry_applet.html
   - favicon.png
   - banner.png
   - imsmanifest.xml
2. Zip the folder.
3. Upload to Moodle as a SCORM package.
